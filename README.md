# Homebrew crudo

Homebrew tap for [crudo](https://github.com/melonask/crudo).

## Install

```sh
brew tap melonask/crudo
brew install crudo
```

Alternatively, install it in one command:

```sh
brew install melonask/crudo/crudo
```

Install the current development version with:

```sh
brew install --HEAD melonask/crudo/crudo
```
